How To Make a File Hierarchy in Adobe Brackets

How to Create An Index.Html

The first and most important step is to create an index.html

Step 1:

Right Click And click "new folder"

Step 2:

Make a new text document and Name the file "Index.Html" MAKE SURE TO CHANGE ".txt" TO ".HTML"

Step 3:

Open it in adobe brackets and copy the following:

<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>Title of the document</title> </head> <body> Content of the document...... </body> </html>

How to add an external a CSS Folder

Step 1:

Create a new folder in the same directory as the index.html name it "css"

Step 2:

Open the "css" folder and make a new text document

Rename it to "styles.css"MAKE SURE TO CHANGE ".txt" TO ".CSS"

Step 3:

Copy and paste the following under the <head> tag in your index.html

<link rel="stylesheet" type="text/css" href="CSS/Styles.css" media="screen" />

How to Create an Assets Folder

Step 1:

Create a New folder called "assets" In the Website's Directory

Step 2:

Paste any images/other assets in the folder

Step 3:

to use this assets use the following

src="/assets/name.placeholder"

How to Create an External JavaScript file

Step 1:

Create a new folder called "javascript in the same directory as your Index.Html

Step 2:

Create a new text file and rename it to "Script.js"MAKE SURE TO CHANGE ".txt" TO ".JS"

Step 3:

Add the following to your "index.html" under the <head> tag

<script src="javaScript/Script.js"></script>
